## COMANDOS BASICOS VI
* Listar el contenido del repositorio de git:

 `git ls-tree master`

 `git ls-tree master^^^`

 `git ls-tree master^3`

* Log en una linea:

 `git log --oneline`

* Log con los tres ultimos commits en una linea:

 `git log --oneline -3`

* Para mas opciones consultar documentacion de git.
